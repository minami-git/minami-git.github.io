## 便利なキー

- 画面を右に表示する: `control + option + command + 右`

- 画面を左に表示する: `control + option + command + 左`

- ターミナル(コンソール)を開く: `cmd + j`

- コンソールで　 javascript を実行する
  `3-3-3.js` を実行したいとき
  ```bash
  node 3-2-3.js [enterキー]
  ```
